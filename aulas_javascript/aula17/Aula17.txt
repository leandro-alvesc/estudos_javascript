Introdução à objeto

No array utilizamos 'let num = [5, 8, 4]', para objetos podemos utilizar chaves, exemplo:
let amigo = {
    nome:'José',    //atributo com o valor
    sexo:'M',
    peso:'85.4',
    engordar(p){}   //pode criar um item com uma função dentro
}
